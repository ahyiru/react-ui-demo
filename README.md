# react-ui-demo
#### 基于react，使用es6编写，webpack构建的 react-ui template

## Getting start
1. Clone this repo:
`
$ git clone https://github.com/ahyiru/react-ui-demo.git
`

2. Install dependenices:
`
$ npm i
`

3. Start the project:
`
$ npm start 		//默认端口8000
`

4. 打开浏览器，访问: [http://localhost:8000/](http://localhost:8000/)

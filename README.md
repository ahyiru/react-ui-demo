## Getting start

1. Clone this repo:

		$ git clone https://github.com/ahyiru/react-ui-demo.git

2. Install dependenices:
`
$ npm i
`

3. Start the project:
`
$ npm start 		//默认端口8000
`

5. 打开浏览器，访问: [http://localhost:9000/](http://localhost:9000/)

## 快速创建项目

1. 安装 `yrui-init` :

		$ npm install -g yrui-init

2. 创建项目:
`
$ yrui-init appName
`

3. 安装依赖:
`
$ cd appName && yarn install
`

4. 运行项目:
`
$ npm start
`
